## How To Run Application

Clone the project

```bash
  git clone https://github.com/sarah-adewale/hubsforjobs-webscrapper
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Start Tailwind

```bash
  npm run tailwind:css 
```
